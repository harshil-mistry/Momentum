### The Secret Sauce (Tech Stack)

This project is a full-stack MERN application.

* **Frontend:** React.js

* **Backend:** Node.js & Express

* **Database:** MongoDB

* **Authentication:** JSON Web Tokens (JWT)

### How to Get Started (The 5-Minute Setup)

1.  **Clone the Repo:**
    ```sh
    git clone https://github.com/harshil-mistry/Momentum.git
    ```

2.  **Set Up the Backend:**
    ```sh
    cd Momentum/backend
    npm i
    # You'll need to create a default.json file in the config folder for your MongoDB URI and JWT Secret!
    ```

3.  **Set Up the Frontend:**
    ```sh
    cd ../frontend
    npm install
    ```

4.  **Run both Servers:**
    ```sh
    cd ..
    npm install
    npm start
    ```
